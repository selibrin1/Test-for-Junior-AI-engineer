# Тестовое задание на позицию Junior AI-engineer

Финансовые аналитики тратят значительное время на изучение отчетов компаний, извлечение ключевых метрик и ответы на вопросы о финансовых показателях.

**Твоя задача — создать прототип AI-ассистента**, который помогает анализировать финансовые данные и отвечать на вопросы о результатах деятельности компании. 

Ассистент должен уметь читать структурированные финансовые данные и генерировать содержательные выводы.

## Цель задания

Создать небольшую систему, которая:
1. Загружает финансовые данные компании
2. Позволяет пользователю задавать вопросы о финансовых показателях
3. Использует LLM для генерации аналитических выводов
4. Поддерживает простые запросы финансового анализа

## Исходные данные 
В репозитории находится файл [financial_data.csv
](https://github.com/HR-amocrm/Test-for-Junior-AI-engineer/blob/b99d291b6385e127ac5009e2994cfd1919456ad3/financial_data.csv) с финансовыми показателями компании за 20 лет (2005-2024).

### Структура данных
| Поле | Описание |
|------|----------|
| `year` | Год отчета |
| `revenue` | Выручка компании ($) |
| `cogs` | Себестоимость продаж (Cost of Goods Sold, $) |
| `operating_expenses` | Операционные расходы ($) |
