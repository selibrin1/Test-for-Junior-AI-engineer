# Тестовое задание на позицию Junior AI-engineer

Финансовые аналитики тратят значительное время на изучение отчетов компаний, извлечение ключевых метрик и ответы на вопросы о финансовых показателях.

**Твоя задача — создать прототип AI-ассистента**, который помогает анализировать финансовые данные и отвечать на вопросы о результатах деятельности компании. 

Ассистент должен уметь читать структурированные финансовые данные и генерировать содержательные выводы.

## Цель задания

Создать прототип, который:
1. Загружает финансовые данные компании
2. Позволяет пользователю задавать вопросы о финансовых показателях
3. Использует LLM для генерации аналитических выводов
4. Поддерживает простые запросы финансового анализа

## Исходные данные 
В репозитории находится файл [financial_data.csv
](https://github.com/HR-amocrm/Test-for-Junior-AI-engineer/blob/b99d291b6385e127ac5009e2994cfd1919456ad3/financial_data.csv) с финансовыми показателями компании за 20 лет (2005-2024).

### Структура данных
| Поле | Описание |
|------|----------|
| `year` | Год отчета |
| `revenue` | Выручка компании ($) |
| `cogs` | Себестоимость проданных товаров или услуг (Cost of Goods Sold, $) |
| `operating_expenses` | Операционные расходы ($) |
| `net_income` | Чистая прибыль ($) |

### Фрагмент данных
```csv
year,revenue,cogs,operating_expenses,net_income
2005,120000,50000,40000,30000
2006,155000,65000,52000,38000
...
2024,3650000,1533000,1205000,912000
```

## Задание 
### 1. Обработка финансовых данных
Необходимо загрузить датасет и рассчитать ключевые финансовые показатели:
- рост выручки (Revenue Growth)
- операционная маржа (Operating Margin)
- чистая маржа (Net Margin)
### 2. Интеграция LLM
Используйте API любой LLM-модели для генерации аналитических выводов.
Модель должна:
- рассуждать на основе финансовых метрик
- генерировать понятные объяснения 
- **не придумывать числовые значения**, которых нет в данных
### 3. Проектирование промптов
Твои промпты должны направлять модель:
- использовать только предоставленные данные
- объяснять расчеты
- выдавать структурированные ответы
### 4. Интерфейс взаимодействия с AI-ассистентом
Реализуйте простой интерфейс, через который пользователь сможет задавать вопросы AI-ассистенту о финансовых данных компании. Интерфейс должен позволять:
- ввести вопрос пользователя
- получить и отобразить ответ ассистента

## Функциональные требования
Система должна поддерживать следующие типы запросов. 
### 1. Аналитические вопросы о финансовых показателях

Пример запроса:

Вопрос:

> В каком году был самый быстрый рост выручки?

Ожидаемый ответ:

> Наибольший рост выручки наблюдался в 2022 году — рост составил 40% по сравнению с предыдущим годом.

### 2. Анализ трендов

Пример запроса:

Вопрос:

> Как изменялась прибыльность компании со временем?

Ожидаемый ответ:

> Прибыльность компании постепенно росла с 2020 по 2023 год. Чистая прибыль увеличилась с $300 тыс. до $830 тыс.

### 3. Объяснение финансовых метрик

Пример запроса:

Вопрос:

> Объясните динамику операционной маржи компании.
